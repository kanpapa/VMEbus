# VME_power

VMEボード P1コネクタのブレイクアウトボードです。VMEボードへの電源の供給およびP1コネクタの信号をすべて引き出せます。

## ボードのアイデア

* VMEbusのP1コネクタから電源（5V, 12V, -12V）を供給します。
* P1コネクタの全信号をコネクタに引き出します。
